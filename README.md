## 📌 Опис проєкту
Проєкт являє собою реалізацію гри **Car Racing** (Складний рівень) мовою Python з використанням бібліотеки `pygame`. 
Метою розробки було застосування базових принципів проєктування (KISS, DRY), налаштування системи контролю версій Git та реалізація перших трьох етапів CI/CD пайплайну: Plan, Code, Build.

## ⚙️ Функціонал гри
* Нескінченний ігровий цикл (Game Loop) із підрахунком балів за час виживання.
* Динамічна генерація ворожих автомобілів.
* **Налаштування через командний рядок (argparse):**
  * `--difficulty` (`easy` / `hard`): Змінює швидкість ворогів, їхню максимальну кількість на екрані та шанс появи (динамічний трафік).
  * `--bg-color` (`gray` / `white` / `black`): Змінює колір дороги та динамічно адаптує колір тексту (UI) для контрастності.

## 🚀 Встановлення та запуск

1. **Клонування репозиторію:**
   ```bash
   git clone [https://github.com/Qu4dyz/car-racing-lab]
   cd SoloProjectCICD

```

2. **Встановлення залежностей:**
Для коректної роботи на сучасних версіях Python використовується Community Edition форк Pygame:
```bash
pip install pygame-ce

```


3. **Запуск гри (стандартні налаштування):**
```bash
python main.py

```


4. **Запуск із кастомними налаштуваннями (додатковий бал):**
```bash
python main.py --difficulty hard --bg-color white

```
